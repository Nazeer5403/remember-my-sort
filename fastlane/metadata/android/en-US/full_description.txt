Persists sort preferences in Android's file picker (DocumentsUI). Manual sort changes are saved and restored across directories and sessions. Defaults to date descending.

<b>Requirements</b>
<ul>
<li>LSPosed framework (API 100+)</li>
<li>Android 11+ (API 30)</li>
<li>Root access</li>
</ul>

<b>Supported Packages</b>
<ul>
<li>com.google.android.documentsui (Google/Pixel devices)</li>
<li>com.android.documentsui (AOSP ROMs)</li>
</ul>

<b>Installation</b>
<ol>
<li>Install LSPosed (JingMatrix fork recommended)</li>
<li>Install this module</li>
<li>Enable in LSPosed Manager</li>
<li>Add DocumentsUI package to module scope</li>
<li>Force stop DocumentsUI</li>
</ol>
